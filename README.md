Gr8ness Company Sales Performance Dashboard
Overview

The Gr8ness Company Sales Performance Dashboard is a business intelligence project developed in Microsoft Excel to analyze and visualize sales performance across multiple business dimensions. The dashboard consolidates raw transactional data into meaningful insights, enabling stakeholders to monitor key performance indicators (KPIs), evaluate sales trends, and support data-driven decision-making.

Designed with an executive audience in mind, the dashboard provides an interactive interface that allows users to filter data by region, year, and payment method through dynamic slicers, making it easier to explore performance from different perspectives.

Project Objectives

This project aims to:

Analyze overall sales performance.
Monitor key business metrics using KPI cards.
Identify revenue-generating countries.
Evaluate customer segment performance.
Analyze monthly sales trends.
Examine shipping mode distribution.
Compare transaction types.
Identify underperforming products.
Present business insights through an interactive dashboard.
